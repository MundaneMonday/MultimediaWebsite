HTML elements

An HTML element is used to denote the different sections of a HTML documents. <html> represents the top level of the Document,or the root element. Inside of the <html> contains the <head> which also contains the meta charset <meta charset="utf-8" /> and the title of the website <title>WEB222 - Assignment 3</title>

The body of the HTML document is denoted by the <body> element. This is where you can put videos, images, links. <div> can separate portions with their own individual id or classes which are referenced in css.
